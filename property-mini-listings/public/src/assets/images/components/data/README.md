# Homes & Beyond Property Listings

## Overview
A Vue 3 single-page property listings application for Homes & Beyond.

## Features
- Dynamic property cards
- Search by title or location
- Sort by price
- Favourite properties
- Availability badges
- Responsive layout

## Installation

1. Clone repository

git clone <repo-link>

2. Install dependencies

npm install

3. Run development server

npm run dev

## Technologies
- Vue 3
- Vite
- CSS
